# CargoVan Connect

This project has been fixed and is ready to run or deploy.

## **How to Run it (Quick Start)**

1.  **Start the Backend**: Double-click `run_backend.bat`. A black window will open. Leave it open.
2.  **Start the Frontend**: Double-click `run_frontend.bat`. A new black window will open, and your browser will launch automatically.

## **How to Submit / Deploy**

Since you need to submit this project, you have two options:

### **Option 1: Submit the Code**
1.  Zip this entire folder.
2.  Submit the zip file.

### **Option 2: Deploy to the Web**
1.  Upload this folder to GitHub.
2.  Go to [Render.com](https://render.com) and create a new Blueprint.
3.  Connect your repository.
4.  The system will automatically use the `render.yaml` file to deploy your site.

## **Technical Details (For Graders/Reviewers)**
- **Backend**: Python (FastAPI). Uses SQLite database.
- **Frontend**: React.
- **Deployment**: Configured for Render via `render.yaml`.
